# ChatIO
Mini Chat Room using Socket.io

## Demo
Check a live demo here: https://guarded-dusk-11495.herokuapp.com/

## Installation

Use the package manager [npm] to install.

```bash
npm install --save
```

## Features

* Chat room
* Enter your nickname
* Realtime messages
* Notfication when new user enters or leaves the room

## Used Packages

* Express
* Socket.io
* Moment
